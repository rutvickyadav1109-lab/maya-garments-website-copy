## My Prompt Framework Logic

Every prompt I wrote followed this structure:

Role → I am working on... (first person)
Business Context → Name, location, brand, collections
Target Audience → Who shops here
Tone → Warm, local, community feel
Output Format → Exactly what sections I need
Rules → Avoid generic phrases, sound human

## Key Learnings

- Specific prompts give better output than generic ones
- First-person prompts feel more authentic
- Adding business USPs in prompt improves AI output
- Same framework works across different industries
